## 一、系统概述

### 1.1 系统定位

Calmara是一个面向校园心理健康服务的智能对话系统，集成了多模态情绪分析、知识检索增强生成(RAG)、多Agent协作、大语言模型微调等先进技术，为学生提供专业的心理健康咨询和情绪支持服务。

### 1.2 核心价值

- **智能化服务**: 7x24小时智能心理咨询服务
- **多模态感知**: 文本、语音、视频多维度情绪分析
- **专业知识库**: 心理学专业知识驱动的精准回复
- **风险预警**: 实时风险评估与预警机制
- **持续进化**: 自动化模型微调实现持续优化

### 3.3 多模态情绪分析模块

#### 3.3.1 模块概述

| 模块名称 | 多模态情绪分析模块 |
| 所属子系统 | Calmara-multimodal |
| 技术实现 | MediaPipe + Whisper + 融合引擎 |
| 支持模态 | 文本、语音、视频 |

#### 3.3.2 功能清单

| 功能编号 | 功能名称 | 功能描述 | 输入模态 |
|----------|----------|----------|----------|
| EMOTION-001 | 文本情绪分析 | 分析文本情感倾向 | 文本 |
| EMOTION-002 | 语音情绪分析 | 分析语音情感特征 | 音频 |
| EMOTION-003 | 面部表情分析 | 分析面部表情情绪 | 视频 |
| EMOTION-004 | 多模态融合 | 融合多模态情绪结果 | 多模态 |
| EMOTION-005 | 情绪趋势分析 | 分析情绪变化趋势 | 历史数据 |

#### 3.3.3 情绪标签体系

| 情绪类别 | 情绪标签 | 描述 |
|----------|----------|------|
| 正向情绪 | JOY | 快乐、愉悦 |
| 正向情绪 | RELIEF | 宽慰、放松 |
| 正向情绪 | HOPE | 希望、期待 |
| 负向情绪 | ANXIETY | 焦虑、紧张 |
| 负向情绪 | DEPRESSION | 抑郁、低落 |
| 负向情绪 | ANGER | 愤怒、烦躁 |
| 负向情绪 | FEAR | 恐惧、害怕 |
| 负向情绪 | SADNESS | 悲伤、难过 |
| 中性情绪 | NEUTRAL | 平静、中性 |
| 复杂情绪 | CONFUSED | 困惑、迷茫 |

#### 3.3.4 多模态融合算法

```
最终情绪分数 = w1 × 文本情绪 + w2 × 语音情绪 + w3 × 视觉情绪

默认权重配置:
- 文本权重 (w1): 0.5
- 语音权重 (w2): 0.3
- 视觉权重 (w3): 0.2
```

#### 3.3.5 API接口规范

| 接口路径 | 方法 | 描述 | 输入类型 |
|----------|------|------|----------|
| `/api/emotion/analyze/text` | POST | 文本情绪分析 | JSON |
| `/api/emotion/analyze/voice` | POST | 语音情绪分析 | MultipartFile |
| `/api/emotion/analyze/video` | POST | 视频情绪分析 | MultipartFile |
| `/api/emotion/analyze/multimodal` | POST | 多模态融合分析 | Multipart |

#### 3.5.3 RAG工作流程

```
用户查询
    │
    ▼
┌─────────────┐
│ 检索决策器   │ ─── 是否需要检索？
└─────────────┘
    │ (是)
    ▼
┌─────────────┐
│ 查询重写器   │ ─── 优化查询表达
└─────────────┘
    │
    ▼
┌─────────────┐
│ 向量嵌入服务 │ ─── 查询向量化
└─────────────┘
    │
    ▼
┌─────────────┐
│  向量存储    │ ─── 相似度检索 (Top-K)
└─────────────┘
    │
    ▼
┌─────────────┐
│ 结果验证器   │ ─── 过滤低相关性结果
└─────────────┘
    │
    ▼
返回知识文档
```

#### 3.5.4 配置参数

| 参数 | 默认值 | 描述 |
|------|--------|------|
| `embedding.dimension` | 1024 | 向量维度 |
| `retrieval.top-k` | 5 | 返回文档数量 |
| `retrieval.threshold` | 0.7 | 相似度阈值 |
| `rerank.enabled` | true | 是否启用重排序 |

### 3.6 风险评估与预警模块

#### 3.6.1 模块概述

| 预警方式 | 邮件通知、系统告警 |

#### 3.6.2 风险等级定义

| 风险等级 | 编码 | 描述 | 响应措施 |
|----------|------|------|----------|
| 低风险 | LOW | 一般心理困扰 | 正常对话引导 |
| 中风险 | MEDIUM | 明显情绪问题 | 建议寻求专业帮助 |
| 高风险 | HIGH | 严重心理问题 | 触发预警，建议就医 |
| 危机 | CRITICAL | 自伤/自杀风险 | 立即预警，提供援助热线 |

#### 3.6.3 风险评估指标

| 指标类别 | 权重 | 检测内容 |
|----------|------|----------|
| 文本风险 | 0.4 | 自伤关键词、消极表达 |
| 情绪风险 | 0.3 | 极端情绪、情绪波动 |
| 行为风险 | 0.2 | 异常行为描述 |
| 历史风险 | 0.1 | 历史风险记录 |

#### 3.6.4 预警流程

```
风险检测
    │
    ▼
┌─────────────┐

└─────────────┘
    │

    ├── MEDIUM ───▶ 标记记录，建议专业帮助
    │
    ├── HIGH ─────▶ 触发预警
    │                    │
    │                    ▼
    │              ┌─────────────┐
    │              │ 发送通知邮件 │
    │              │ 给心理咨询师 │
    │              └─────────────┘
    │
    └── CRITICAL ─▶ 紧急预警
                         │
                         ▼
                   ┌─────────────┐
                   │ 即时通知    │
                   │ 提供援助热线 │
                   │ 记录存档    │
                   └─────────────┘
```

```yaml
services:

      - mysql

    environment:
      MYSQL_ROOT_PASSWORD: ${DB_PASSWORD}
      
  milvus:
    image: milvusdb/milvus:latest
    ports:
      - "19530:19530"
```
